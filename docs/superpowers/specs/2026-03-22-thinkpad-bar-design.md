# Дизайн-спецификация: ThinkPadBar

**Дата:** 2026-03-23
**Статус:** Реализовано (Core)
**Стек:** Rust, iced (fork with Wayland support), tokio, chrono, sysinfo, system-tray, serde

## 1. Цели проекта
Создать специализированную статусную панель для ноутбуков ThinkPad (целевое устройство: T14 Gen 1 AMD), работающую в среде Hyprland (Wayland). Панель объединяет стандартные функции (время, сеть) с глубоким управлением оборудованием ThinkPad (вентилятор, профили питания, TrackPoint) и интеграцией с системным треем.

## 2. Архитектура (Native Rust)
Панель построена на базе фреймворка **iced** (используется форк с поддержкой Layer Shell) и архитектуры MVU (Model-View-Update).

### Основные компоненты:
*   **Bar App**: Основной цикл приложения, управляющий двумя окнами Layer Shell (основная панель и слой для поп-апов).
*   **Module Engine**: Система модулей в `src/modules/`. Каждый модуль инкапсулирует логику получения данных и выполнения команд.
*   **Popup System**: Многофункциональные всплывающие окна для управления настройками и просмотра детальной статистики.

## 3. Модули и функциональность

### Группа "ThinkPad Hardware"
1.  **Fan Control**: Индикация RPM и выбор режима (Auto, 0-7, Full Speed).
2.  **Power Management**: Управление профилями питания (Performance, Balanced, Power Saver) и отображение статуса батареи (заряд, потребление).
3.  **Keyboard Layout**: Отображение текущей раскладки и переключение кликом.

### Группа "System & Network"
4.  **System Monitor**: Мониторинг CPU (%), RAM (%), Swap (%), Temp (°C), Disk usage, Network speed (Up/Down).
5.  **Connectivity**:
    *   **WiFi**: Статус, сканирование сетей, подключение с вводом пароля через GUI.
    *   **Bluetooth**: Отображение статуса и переключение (Toggle).
6.  **Audio & Mic**: Управление громкостью через слайдеры, быстрое переключение Mute.
7.  **Brightness**: Слайдер управления яркостью экрана.

### Группа "Desktop"
8.  **Workspaces**: Полная интеграция с Hyprland (отображение активных/занятых воркспейсов, переключение кликом).
9.  **System Tray**: Полноценная поддержка StatusNotifierItem (иконки приложений в трее с возможностью активации).
10. **Calendar**: Всплывающий календарь с навигацией по месяцам.

## 4. Визуальный дизайн (Tokyo Night)
*   **Фон:** `#1a1b26` (прозрачность 85-95%) для элементов, `#11121d` для поп-апов.
*   **Акцент:** `#7aa2f7` (Blue) для активных элементов.
*   **Статус:** `#9ece6a` (Green), `#e0af68` (Yellow), `#f7768e` (Red) для температур и критических уровней.
*   **Шрифт:** JetBrains Mono Nerd Font.

## 5. Технические детали
*   **Layer Shell**: Используется `iced` с поддержкой `layer-surface-settings`.
*   **D-Bus**: Используется для управления сетевыми подключениями и треем.
*   **SysFS/ProcFS**: Прямое взаимодействие с `/proc/acpi/ibm/` для специфичных функций ThinkPad (кулеры).

## 6. Текущий статус
Проект находится в стадии активного использования и полировки UI. Базовый функционал полностью реализован.
